# AGENTS.md

Project: NDMII Platform

This project builds a government-grade MVP platform for Nigeria’s Digital MSME Identity Infrastructure Initiative.

Codex should:

Prioritize working functionality over theoretical architecture

Build a premium civic-tech UI suitable for presentation to federal regulators

Support these user roles:

Public
MSME
Association Officer
Reviewer
FCCPC Officer
FIRS Officer
Admin

Core modules required:

Landing site
MSME onboarding wizard
Identity generation engine
QR verification portal
KYC simulation module
Complaint module
Tax/VAT simulation module
Manufacturer traceability module
Analytics dashboards
Export/reporting module

External integrations must be simulated:

NIN
BVN
CAC
TIN

These integrations should use adapter services that can later be replaced with live APIs.

System must:

be responsive
use reusable components
use realistic Nigerian demo data
generate QR codes
generate MSME IDs automatically
support role-based dashboards

Avoid unnecessary complexity.

Focus on end-to-end working system delivery.
